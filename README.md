# Finscale Media — Landing Page

Site éditorial Finscale : podcast finance & fintech animé par Solenne Niedercorn.

## Structure

```
index.html          — Page principale (HTML/CSS standalone)
assets/             — Logo, favicon, photo Solenne, thumbnails épisodes
```

## Déploiement

Site 100% statique. Aucune dépendance backend.

### GitHub Pages
1. Settings → Pages
2. Source : `Deploy from a branch` → `main` → `/root`
3. Live sur `https://nn70007.github.io/finscale_media/`

### Netlify / Vercel
Import depuis GitHub → détection auto → live en 30s.

## Stack

- HTML5 / CSS pur
- Google Fonts : IBM Plex Serif, IBM Plex Sans, Playfair Display
- Vanilla JS (menu mobile, progress bar, banner cookies)
